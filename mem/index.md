# Project Memory

## Core
Product is a Creative Studio — a workflow & creative-cognition tool for nonlinear thinkers, founders, ND-friendly creators. NOT an assessment, NOT psychological, NOT diagnostic.
Everything is user-selected. Never infer, analyze, score, or interpret the user.
Forbidden vocab: assessment, score, dimension, profile, trait, symptom, diagnosis, Dąbrowski, overexcitabilities, OE, 2E, intensities, developmental level, mental health, wellness.
Categories are symbolic: Flux / Depth / Signal / Myth / Pulse Modes. Phases: Initiation / Expansion / Integration / Synthesis (non-hierarchical, story-based).
Aesthetic: atmospheric, mythic-tech, ND-friendly, indigo twilight palette, Cormorant Garamond display + Inter body.
Local-first storage (studioStore, key creative-studio:v1). Cloud-ready later.
Replace certainty with exploration; labels with narratives; prescriptions with experiments.
